# 🏦 Banco-CrediNova

¡Bienvenido a **Banco-CrediNova**! Este es el sitio web oficial de una banca en línea interactiva y moderna. El proyecto simula las operaciones financieras esenciales de una entidad bancaria real, ofreciendo una experiencia visual limpia y estructurada directamente en el navegador.

## ✨ Características Principales
* **Autenticación de Usuario:** Sistema de inicio de sesión y registro de nuevas cuentas en tiempo real.
* **Incentivo de Apertura:** Asignación automatizada de fondos iniciales aleatorios (con un tope máximo de Q50,000.00 por cuenta) expresados en Quetzales (Q).
* **Transferencias Directas:** Módulo de transacciones seguras e inmediatas entre cuentas de usuarios registradas.
* **Módulo QR Express:** Permite la generación de tokens encriptados en formato `Base64` que emulan el comportamiento de un código QR real para cobros y pagos rápidos.

## 🛠️ Tecnologías Utilizadas
* **HTML5:** Estructura semántica del sitio web.
* **CSS3:** Diseño visual personalizado con la identidad corporativa oficial (Verde Oscuro y Café).
* **JavaScript (Vanilla):** Lógica del negocio, manejo de base de datos simulada en memoria y encriptación de tokens QR.

## 🎨 Identidad Visual
* **Colores Oficiales:** Verde Oscuro (`#1b4332`) y Café Tierra (`#4e3629`), seleccionados para transmitir solidez, crecimiento y seguridad financiera.
